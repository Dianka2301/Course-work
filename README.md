# Food Recipe App

Food Recipe App — це сучасна кулінарна веб-платформа, що поєднує зручний робочий простір для щоденного готування, соціальну взаємодію між користувачами та інструменти штучного інтелекту (Google Gemini API).

Проєкт розроблено за архітектурою клієнт-сервер (React + Node.js) із використанням локальної бази даних SQLite.

# Налаштування та запуск

## 1. Системні вимоги
- Node.js (версія 16 або вище)
- npm

---

## 2. Backend

```bash
cd backend
npm install
```

Створіть `.env` файл у backend:

```env
PORT=4000
JWT_SECRET=ваша_секретна_строка_jwt
GEMINI_API_KEY=ваш_api_ключ_від_gemini
```

Запуск backend:

```bash
node server.cjs
```

Сервер буде доступний:

```text
http://localhost:4000
```

---

## 3. Frontend

```bash
cd frontend
npm install
```

Створіть `.env` у frontend:

```env
VITE_API_URL=http://localhost:4000
```

Запуск frontend:

```bash
npm run dev
```

Додаток буде доступний:

```text
http://localhost:5173
```

## Тестування

У проєкті реалізовано інтеграційне тестування backend-частини із використанням:

- Jest  
- Supertest  
- JWT для авторизованих запитів  
- окремої тестової SQLite бази даних  

### Тестове покриття

Тести покривають такі функціональні модулі:

- реєстрацію та авторизацію користувачів  
- перевірку JWT-аутентифікації  
- отримання та оновлення профілю  
- створення приватних рецептів  
- роботу каталогу рецептів  
- систему обраного (favorites)  
- коментарі та вкладені відповіді  
- систему рейтингів  
- пагінацію рецептів  
- модерацію рецептів адміністратором  
- генерацію сповіщень  
- перевірку схожих рецептів за інгредієнтами  

### Тестова база даних

Для тестування використовується ізольована база даних `test-database.db`, яка автоматично створюється перед запуском тестів та видаляється після їх завершення.
