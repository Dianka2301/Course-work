# Food Recipe App

Food Recipe App — це сучасна кулінарна веб-платформа, що поєднує зручний робочий простір для щоденного готування, соціальну взаємодію між користувачами та інструменти штучного інтелекту (Google Gemini API).

Проєкт розроблено за архітектурою клієнт-сервер (React + Node.js) із використанням локальної бази даних SQLite.

---

# Основні можливості

## Профіль користувача
- Редагування імені, прізвища, email та завантаження аватара (з кроппером зображень)
- Додавання біографії (біо), що синхронізується з базою даних
- Перегляд публічних профілів інших користувачів через сторінку автора (фото, біо, рецепти)

---

## Інтерактивний каталог та обране
- Категорії з горизонтальним скролом (адаптивність: мобільні / планшети)
- Пошук за назвою або інгредієнтами
- Сортування: рейтинг, новизна, складність (easy / medium / hard)
- Пагінація (15 рецептів на сторінку)
- Додавання в обране через кнопку ❤️

---

## Спільнота та обговорення
- Вкладені коментарі (replies)
- Оцінки рецептів
- Автоматичне вирівнювання дат для чистого UI
- AI-фільтр коментарів за кулінарною корисністю

---

## AI-генератор рецептів
- Генерація рецептів за інгредієнтами
- Розрахунок КБЖУ, часу та складності
- Історія згенерованих рецептів

---

## Записник та модерація
- Приватні рецепти користувача
- Завантаження фото
- AI-модерація перед публікацією (перевірка якості, спаму, небезпечних інгредієнтів)

---

## Адмін-панель
- Split-панель для заявок
- AI-аналіз рецептів
- Рішення: схвалити / відхилити

# Налаштування та запуск

## 1. Системні вимоги
- Node.js (версія 16 або вище)
- npm

---

## 2. Backend

```bash
cd backend
npm install
```

Створіть .env файл у backend:

```PORT=4000
JWT_SECRET=ваша_секретна_строка_jwt
GEMINI_API_KEY=ваш_api_ключ_від_gemini```

Запуск backend:

```node server.cjs```

Сервер буде доступний:

```http://localhost:4000```

3. Frontend
```cd frontend
npm install
```

Створіть .env у frontend:

```VITE_API_URL=http://localhost:4000```

Запуск frontend:

```npm run dev```

Додаток буде доступний:

http://localhost:5173
