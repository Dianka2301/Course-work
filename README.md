# Food Recipe App

Food Recipe App — курсова веб-робота з різними цікавими рецептами та фічами.  
Проєкт побудований як кулінарна книга з фронтендом і бекендом, з використанням JavaScript та простих технологій.  

Особливості:
- Початкова сторінка з входом та реєстрацією
- AI-помічник, який генерує рецепти з обраних продуктів
